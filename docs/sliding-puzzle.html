Comment jouer au Taquin

Le Taquin utilise une des couvertures de vos livres comme image a reconstituer. Faites glisser les tuiles pour retrouver l'image complete.

3 niveaux de difficulte

Niveau Grille Tuiles
Facile 3x3 8
Moyen 4x4 15
Difficile 5x5 24

Comment jouer

  1. Choisissez un niveau de difficulte
  2. Une couverture de votre bibliotheque est selectionnee au hasard
  3. L'image est decoupee en tuiles avec un espace vide
  4. Touchez une tuile adjacente a l'espace vide pour la faire glisser
  5. Placez toutes les tuiles dans le bon ordre pour terminer le puzzle
  6. Une image de reference est affichee pour vous aider

Score

Votre score est base sur le nombre de coups et le temps ecoule. Moins de coups et un temps plus rapide donnent un meilleur score.

Classement

Vos meilleurs scores sont enregistres pour chaque niveau. Tentez de battre vos propres records !
