Libérez votre bibliothèque

Vos livres ne doivent pas être prisonniers d'une plateforme.

BiblioGenius est conçu pour respecter votre souveraineté numérique. Voici comment récupérer vos données depuis les plateformes les plus courantes pour les importer chez vous.

Voir le tutoriel vidéo

📱 Gleeph

Exportez votre bibliothèque au format CSV :

  • Ouvrez l'application Gleeph sur votre mobile.
  • Allez dans l'onglet Profil.
  • Appuyez sur la roue dentée (Paramètres) en haut à droite.
  • Sélectionnez Exporter mes données.
  • Vous recevrez un fichier CSV par email ou pourrez l'enregistrer directement.

📖 Babelio

Récupérez votre liste de lecture :

  • Connectez-vous à votre compte sur le site Babelio.
  • Allez dans Mes livres.
  • Cherchez le lien Exporter mes livres (souvent en bas de page ou dans les options de liste).
  • Téléchargez le fichier CSV généré.

🌎 Goodreads

L'export standard de Goodreads :

  • Allez sur Goodreads.com et connectez-vous.
  • Cliquez sur My Books dans la barre de navigation.
  • Dans le menu de gauche, cliquez sur Import and Export.
  • Cliquez sur le bouton Export Library en haut à droite.
  • Attendez que le fichier soit généré et cliquez sur le lien de téléchargement.

📦 inventaire.io

Le projet ami respectueux des données :

  • Connectez-vous à inventaire.io.
  • Allez dans vos Paramètres de profil.
  • Recherchez la section Données ou Export.
  • Choisissez le format CSV ou JSON (BiblioGenius préfère le CSV pour le moment).

Et ensuite ?

Une fois votre fichier en main, ouvrez BiblioGenius et utilisez l'assistant de migration pour importer vos livres en un clic.
