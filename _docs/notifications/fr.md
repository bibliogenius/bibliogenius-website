L'ecran Notifications centralise tous les evenements lies a votre reseau : demandes de connexion, emprunts, retours de livres et correspondances avec votre liste de souhaits.

## Types de notifications

- **Demande de connexion** : un lecteur souhaite se connecter a votre bibliotheque
- **Connexion acceptee** : votre demande de connexion a ete acceptee
- **Demande d'emprunt** : quelqu'un souhaite emprunter un de vos livres
- **Emprunt accepte** : votre demande d'emprunt a ete acceptee
- **Livre retourne** : un emprunteur a rendu votre livre
- **Correspondance souhait** : un livre de votre liste de souhaits a ete trouve dans la bibliotheque d'un contact

## Filtrer les notifications

En haut de l'ecran, des filtres permettent d'afficher uniquement certaines categories :

- **Tout** : toutes les notifications
- **Connexions** : demandes et acceptations de connexion
- **Emprunts** : demandes, acceptations et retours de livres
- **Decouvertes** : correspondances avec votre liste de souhaits

## Actions

- **Appuyer sur une notification** : ouvre l'ecran correspondant (reseau, demandes, bibliotheque du contact)
- **Glisser vers la gauche** : supprime la notification
- **Tout marquer comme lu** : via le bouton dans la barre d'actions
- **Tout supprimer** : via le bouton dans la barre d'actions (avec confirmation)
