L'écran Notifications centralise tous les événements liés à votre réseau : demandes de connexion, emprunts, retours de livres et correspondances avec votre liste de souhaits.

## Types de notifications

- **Demande de connexion** : un lecteur souhaite se connecter à votre bibliothèque
- **Connexion acceptée** : votre demande de connexion a été acceptée
- **Demande d'emprunt** : quelqu'un souhaite emprunter un de vos livres
- **Emprunt accepté** : votre demande d'emprunt a été acceptée
- **Livre retourné** : un emprunteur a rendu votre livre
- **Correspondance souhait** : un livre de votre liste de souhaits a été trouvé dans la bibliothèque d'un contact

## Filtrer les notifications

En haut de l'écran, des filtres permettent d'afficher uniquement certaines catégories :

- **Tout** : toutes les notifications
- **Connexions** : demandes et acceptations de connexion
- **Emprunts** : demandes, acceptations et retours de livres
- **Découvertes** : correspondances avec votre liste de souhaits

## Actions

- **Appuyer sur une notification** : ouvre l'écran correspondant (réseau, demandes, bibliothèque du contact)
- **Glisser vers la gauche** : supprime la notification
- **Tout marquer comme lu** : via le bouton dans la barre d'actions
- **Tout supprimer** : via le bouton dans la barre d'actions (avec confirmation)
