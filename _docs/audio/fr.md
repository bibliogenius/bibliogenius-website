Le module Audio recherche automatiquement des versions audio gratuites de vos livres sur des banques en ligne libres : LibriVox, Litteratureaudio.com et Internet Archive. Activez le module dans les parametres, puis ecoutez directement depuis l'application.

## Activer le module Audio

1. Allez dans votre Profil
2. Ouvrez les parametres
3. Activez le module "Audio"

![Parametres des modules](images/settings-modules.png)

## Sources disponibles

- **LibriVox** : le plus grand catalogue de livres audio libres, en plusieurs langues
- **Litteratureaudio.com** : classiques francais en haute qualite
- **Internet Archive** : catalogue tres large, qualite variable

## Ecouter un livre audio

Quand un livre audio gratuit est disponible pour un livre de votre bibliotheque, BiblioGenius le detecte automatiquement. Vous pouvez alors l'ecouter directement depuis la fiche du livre, chapitre par chapitre.
