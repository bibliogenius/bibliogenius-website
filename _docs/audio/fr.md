Le module Audio recherche automatiquement des versions audio gratuites de vos livres sur des banques en ligne libres : LibriVox, Litteratureaudio.com et Internet Archive. Activez le module dans les paramètres, puis écoutez directement depuis l'application.

## Activer le module Audio

1. Allez dans votre Profil
2. Ouvrez les paramètres
3. Activez le module "Audio"

![Paramètres des modules](images/settings-modules.png)

## Sources disponibles

- **LibriVox** : le plus grand catalogue de livres audio libres, en plusieurs langues
- **Litteratureaudio.com** : classiques français en haute qualité
- **Internet Archive** : catalogue très large, qualité variable

## Écouter un livre audio

Quand un livre audio gratuit est disponible pour un livre de votre bibliothèque, BiblioGenius le détecte automatiquement. Vous pouvez alors l'écouter directement depuis la fiche du livre, chapitre par chapitre.
