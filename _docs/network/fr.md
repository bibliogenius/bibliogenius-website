La section "Reseau" vous permet de decouvrir et consulter les catalogues de bibliotheques ajoutees. Vous pouvez y faire des demandes d'emprunt.

![Vue du reseau](images/network-view.png)

## Parcourir un catalogue

Selectionnez une bibliotheque dans votre reseau pour voir les livres disponibles. Vous pouvez rechercher un titre precis ou parcourir les etageres.

## Demander un emprunt

Quand vous trouvez un livre qui vous interesse, appuyez dessus pour envoyer une demande d'emprunt au proprietaire.
