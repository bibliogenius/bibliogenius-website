La section "Réseau" vous permet de découvrir et consulter les catalogues de bibliothèques ajoutées. Vous pouvez y faire des demandes d'emprunt.

![Vue du réseau](images/network-view.png)

## Parcourir un catalogue

Sélectionnez une bibliothèque dans votre réseau pour voir les livres disponibles. Vous pouvez rechercher un titre précis ou parcourir les étagères.

## Demander un emprunt

Quand vous trouvez un livre qui vous intéresse, appuyez dessus pour envoyer une demande d'emprunt au propriétaire.
