The "Network" section lets you discover and browse catalogs of libraries you've added. You can request to borrow books from there.

![Network view](images/network-view.png)

## Browse a catalog

Select a library in your network to see available books. You can search for a specific title or browse through shelves.

## Request a loan

When you find a book you're interested in, tap it to send a borrow request to the owner.
